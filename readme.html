Getting Started with StoreFrontal

StoreFrontal Theme is a beautifully-designed WordPress e-Commerce theme with easy-to-use theme options and a light and dark color scheme. The theme was designed by acclaimed designer Vin Thomas from Fixel

Installing StoreFrontal

  1. Automatic: Go to Dashboard > Appearance > Themes > Install Themes tab > Upload, and use the upload form.
  2. Manual: Using the FTP client of your choice, unzip and install the StoreFrontal theme into the /wp-content/themes/ directory.

Setting Up StoreFrontal

Installing Required and Recommended Plugins

StoreFrontal requires or recommends several Plugins for optimal functionality. All of these Plugins are available in the official WordPress Plugin repository, and can be installed via Dashboard > Plugins > Add New, by searching for the Plugin name.

  1. StoreFrontal requires the following Plugins for proper functionality:
    1. WP PageNavi: Numerical pagination links
    2. MediaElement.js: HTML5 audio/video player
    3. ShareThis: Social network sharing links
  2. StoreFrontal recommends the following Plugins for optimal functionality:
    1. WP e-Commerce: e-commerce Plugin
    2. Widget Logic: control Widget display using WordPress contextual conditional tags
  3. Install the latest version of the WP E-commerce plugin.

Setting Up StoreFrontal

  1. Menus: StoreFrontal supports one navigation menu: the Primary Navigation that displays at the top of the page. Once you have added your static Pages, go to Dashboard > Appearance > Menus to create one or more custom menus. Then, select one of your menus in the "Primary Navigation" dropdown in the Theme Locations meta box in the upper left-hand column.
  2. Widget areas: Go to Dashboard > Appearance > Widgets to configure Widgets for the Theme. StoreFrontal supports several Widget areas:
    1. Default Sidebar: the sidebar that displays on the blog posts index and single blog posts
    2. Footer 1 (left side): Left-hand column footer Widget area
    3. Footer 2 (right side): Right-hand column footer Widget area
  3. Custom Widgets: StoreFrontal includes some custom Widgets that you may find useful, including:
    1. Post Formats: Widget to display a list of the most recent posts with the specified post format
  4. Theme Options: Go to Dashboard > Appearance > Theme Options to configure the StoreFrontal Theme options.

Setting Up your Site Front Page

  1. Go to Dashboard > Pages > Add New to create a new static Page. Give it whatever title you want, such as "Home".
  2. Under the Page Attributes meta box in the right column of the Edit Page screen, in the Template dropdown, select the Home page template.
  3. Go to Dashboard > Settings > Reading and change "Front Page Displays" to "A static page".
  4. In the "front page" dropdown, select the page you just created.
  5. (Note: if you intend to have a blog, create another static page (using the default template), and select it under the "posts page" dropdown.)
  6. Next, go to Dashboard > Appearance > Widgets to configure the front page Widgets. The front page includes three Widget areas:
    1. Column 1 (homepage): left-hand column Widget area
    2. Column 2 (homepage): center column Widget area
    3. Column 3 (homepage): right-hand column Widget area
